# 01 — Problem Definition / 问题定义

## 初始问题

跨国旅行者常同时携带：

- GaN 充电器；
- 地区转换插头或万能转换头；
- 充电宝；
- 多根线材；
- 手表、相机等专用充电附件。

传统产品主要竞争：

- 最大功率；
- 端口数量；
- 覆盖国家数量；
- 体积和重量。

## 第一轮证据提示的问题

公开用户讨论和个人旅行经历提示，旅行供电失败不只来自“功率不足”，还来自：

1. 重型充电器与转换头叠加后产生更大的墙端下坠力矩；
2. 老旧、松动、凹陷或被家具遮挡的插座无法稳定承载；
3. 万能转换头存在体积、插脚、接触和地区兼容性差异；
4. 用户睡前看到“已连接”，但无法确认整夜是否持续充电；
5. 旅行者可能携带大量冗余附件，但“模块化”本身也会增加复杂度。

## Jobs to Be Done

目标用户不是“所有旅行者”，而是：

> 高频跨国、多设备、对次日设备就绪高度敏感的旅行者。

核心任务：

> 在陌生插座条件下，以尽可能少的配置错误和机械失效，稳定完成夜间多设备补能，并让用户在早晨前确认关键设备已经就绪。

## 非目标

当前阶段不试图：

- 覆盖所有国家和所有高功率家电；
- 充当电压转换器；
- 证明真实掉落率或真实市场份额；
- 强迫用户安装 App；
- 将所有配件都做成模块；
- 直接预测用户愿意支付的溢价。

## 当前设计原则

1. **重的部分不挂墙**；
2. **地区适配功能与高功率电子功能解耦**；
3. **本机优先确认，App 仅作可选增强**；
4. **模块数量少于功能数量**；
5. **每个新增模块必须证明其减少的负担大于新增的复杂度**。
