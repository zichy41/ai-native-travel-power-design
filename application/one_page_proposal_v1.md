# Atlas Relay — One-Page Proposal

## Problem
重型 GaN 充电器与旅行转换头叠加后，在松动、凹陷或受遮挡插座中可能发生脱落、接触中断和整夜未充入电。现有产品主要优化国家覆盖、峰值功率和端口数量，较少把墙端机械可靠性与供电事件确认作为统一目标。

## Target User
高频跨国、多设备旅行者：携带笔记本，同时携带手机和至少一台相机、平板或配件，对夜间集中补能可靠性敏感。

## Product
**Anker Atlas Relay**
- 地区短电源线；
- 100W 3C GaN 桌面主机；
- AC 中断与恢复记录；
- USB-C 重连和异常低功率记录；
- 每端口功率与累计能量；
- 本机状态显示；
- 可选 BLE；
- 无 AC 万能透传；
- 无大容量电池。
