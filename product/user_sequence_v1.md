# Atlas Relay User Sequence

## Before Travel
1. 选择目的地；
2. 系统提示地区线；
3. 只带本次行程所需的一到两条线；
4. 检查设备总功率是否超出 100W 工作范围。

## Hotel Setup
1. 轻量地区插头接墙面；
2. 主机放桌面或床头柜；
3. 接入笔记本、手机和第三台设备；
4. 主机显示总功率和端口分配。

## Overnight
- AC 正常：记录能量和端口状态；
- AC 中断：超级电容保存事件；
- 接触反复恢复：记录 `AC LOST/RESTORED`；
- 异常低功率：标黄；
- 温度异常：降额并记录。

## Morning
用户查看：
- 是否发生异常；
- 哪个端口重连；
- 各端口累计输出能量；
- 是否需要检查线材或插座。

系统不承诺显示所有设备准确电量。
