# Roadmap

## M0 — 立即完成

1. 检查所有图片、CSV 和 Markdown 能否打开；
2. 将 GitHub 链接填入报名表；
3. 提交报名。

## M1 — 入围前自主推进

- 完成 6–10 名真实用户访谈；
- 制作无电机械等质量样机；
- 完成 30/60/90cm 线长概念测试；
- 完成插座保持力、凹陷和遮挡实验；
- 把真实数据放入 `experiments/real/`；
- 更新假设登记表和 Kill Criteria。

## M2 — 入围与导师阶段

- 使用安克内部 AI 工具重跑研究流程；
- 让安克产品、电源、安全团队审查架构；
- 确认目标功率、端口分配、线组和认证边界；
- 将工作流 mock 升级为实际 Agent 原型；
- 比较 AI 驱动与经验驱动产出的差异。

## M3 — 终赛

- 功能演示原型；
- 关键实体实验结果；
- 最终产品提案；
- AI 原生 SOP；
- 3–5 分钟演示视频；
- 答辩 PPT 。
