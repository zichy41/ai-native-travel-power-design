# 06 — Next Steps / 下一步计划

## P0：报名材料前必须完成

- [ ] 将公开用户证据扩展到至少 40 条独立讨论或评论；
- [ ] 编码问题类别：松动、掉落、遮挡、凹陷、过热、插头不兼容、断充、冗余附件；
- [ ] 完成 8–12 个竞品结构拆解；
- [ ] 完成三种候选方案对比；
- [ ] 建立 AI 驱动 vs 经验驱动流程图；
- [ ] 将仓库链接加入报名补充材料。

## P1：真实验证

- [ ] 访谈至少 5 位真实跨国旅行者；
- [ ] 允许无痛点和反方用户进入样本；
- [ ] 记录原话，不诱导“模块化”“短线”“App”；
- [ ] 进行简单概念卡片排序。

## P2：实体工程实验

计划制作三个质量/重心等效模型：

1. 直插充电器；
2. 转换头 + 直插充电器；
3. 轻量插头 + 短线桌面主机。

建议记录：

- 插座类型；
- 墙端质量；
- 重心距离；
- 插入角度；
- 重复次数；
- 轻触扰动；
- 是否发生断连；
- 视频和原始表格。

## P3：AI 工作流原型

计划使用：

- Python 数据清洗与聚类；
- 大模型多角色评审；
- 向量检索与证据引用；
- 飞书多维表格维护假设与决策；
- 简单 Web 页面展示产品定义流程。

## P4：产品概念验证

对至少三个方向评分：

- Atlas Relay：短线高可靠旅行供电；
- 一体式高稳定壁插；
- 仅软件化行程供电配置助手。

统一评分维度：

- 用户痛点强度；
- 现有方案缺口；
- 工程可行性；
- 安克业务适配；
- 差异化；
- 比赛期间可验证性。
