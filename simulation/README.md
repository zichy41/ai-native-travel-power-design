# Torque Simulation / 墙端力矩仿真

## 目的

比较三种结构的静态墙端下坠力矩：

1. 65W 充电器直接插墙；
2. 旅行转换头 + 65W 充电器叠加；
3. 轻量地区插头 + 短线 + 桌面主机。

## 公式

\[
\tau = m g d
\]

对于叠加结构，分别计算转换头和充电器对插座支点的力矩并求和。

## 参数

- 65W 充电器：132g；长度 66.1mm；
- 旅行适配器：107g；厚度 24.9mm；
- 轻量墙端插头：25g；重心距墙 10mm（设计目标假设）。

官方规格来源见 `research/references.md`。

## 运行

```bash
pip install -r requirements.txt
python simulation/torque_model.py
```

## 当前基准结果

- 65W 直插：约 0.043 N·m
- 转换头 + 65W 充电器：约 0.088 N·m
- 轻量插头 + 短线：约 0.0025 N·m
- 静态力矩相对叠加结构下降约 97%

## 重要限制

模型不能预测真实掉落率，因为真实稳定性还取决于：

- 插座弹片夹持力；
- 插脚几何与表面材料；
- 插座磨损；
- 外部扰动和振动；
- 插入深度；
- 线材方向。

因此结果应表述为：

> “短线架构显著降低墙端静态下坠力矩。”

不得表述为：

> “短线架构使掉落率降低 97%。”
