# Repository Map

| 路径 | 内容 | 当前用途 |
|---|---|---|
| `README.md` | 项目总览 | 评委首先阅读 |
| `PROJECT_STATUS.md` | 已完成/未完成 | 目前进度 |
| `ROADMAP.md` | 后续计划 | 展示执行路线 |
| `application/` | 报名文案 | 复制到报名表 |
| `docs/00–06` | 初始问题和 Baseline | 展示方法演化 |
| `docs/07–09` | 工作流 V2、对照协议、第二轮研究 | 核心方法 |
| `docs/10_*` | 当前产品定义 | 当前结论 |
| `research/` | 竞品、公开证据、来源、假设 | 证据层 |
| `workflow/` | Agent 角色、Prompt、Schema | SOP 机器可读层 |
| `prototype/` | 工作流 mock 与静态看板 | 可演示原型 |
| `simulation/` | 静态力矩模型 | 工程预验证 |
| `engineering/` | 系统架构、FMEA、实验 | 可行性与边界 |
| `product/` | 产品外观和结构 | 产品提案 |
| `results/` | 图表、评分和输出 | 结果展示 |
| `scripts/` | 自动检查 | 仓库维护 |
