# Reproducible Test Rig Design V1
# 可复现测试台设计

## 原则

- 可重复、可校准；
- 机械测试与带电测试分离；
- 不依赖“手感”；
- 允许竞争方案公平比较；
- 保存原始视频、传感器数据与元数据。

## 机械测试台

### 底座
- 600 × 400mm 刚性板；
- 插座安装面可在 0–90° 锁定；
- 角度刻度精度至少 1°；
- 插座中心高度统一。

### 可更换插座板
记录：
- 插座类型；
- 品牌/型号；
- 新旧状态；
- 已知插拔循环数；
- 初始拔出保持力；
- 是否仅用于无电机械测试。

破损、改装或人为弯曲触点的插座不得带电。

### 凹陷框与遮挡板
- 10mm 模块化凹陷框，最大 30mm；
- 遮挡板距插座 25–100mm 可调；
- 模拟床头板、桌板和行李架。

## 测量模块

### 拔出保持力
- 数字测力计与插头同轴连接；
- 拉出速度统一；
- 每次确认完全插入；
- 限制同一插座连续循环，避免磨损改变结果。

### 重心与力矩
- 称量墙端实际质量；
- 估计重心；
- 记录重心至墙面的水平距离；
- 多组件分别计算并求和。

### 扰动
- 固定摆锤支架；
- 冲击点和方向统一；
- 每次拍照确认角度；
- 慢动作视频确认首次位移和脱落。

## 带电测试台

### 安全分区

```text
区域 A：市电与 AC 输入（封闭）
区域 B：待测充电器（阻燃台面）
区域 C：USB-C 测量与电子负载
区域 D：计算机和操作人员
```

最低配置：
- 上级漏电保护；
- 过流保护；
- 紧急断电；
- 阻燃外壳；
- 隔离或可编程 AC 测试源；
- 温度异常自动停止；
- 无裸露高压端子。


## 可复现性要求

- 每张图能追溯到原始 CSV；
- 每个结论标明样本数；
- 不修改原始数据；
- 数据清洗脚本进入 GitHub；
- 模拟值和实测值分目录；
- 失败实验也保留。
