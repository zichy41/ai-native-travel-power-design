# Atlas Relay Physical Experiment Protocol V1
# 实体实验方案与可复现测试台设计

> 状态：实验设计稿，尚未代表已完成实体实验。  
> 带电测试必须在具备漏电保护、隔离与消防条件的学校实验室内，由指导人员监督完成。

## 1. 研究问题

本轮实验不试图证明“Atlas Relay 一定成功”，而是检验五个可被否决的问题：

1. 短输入线将主机移离墙面后，是否显著降低墙端机械负载？
2. 相比安克 Prime 一类短机身壁插，短线方案在低夹持力、凹陷和受遮挡插座中是否仍有优势？
3. 30cm、60cm、90cm 三种输入线长度中，哪一种在可达性、收纳和稳定性之间最均衡？
4. AC 中断、USB-C 重新协商和异常低功率事件能否被可靠检测，且误报率可接受？
5. 100W 三 USB-C 主机在多设备负载下，是否可以保持合理温升并避免非必要端口中断？

## 2. 总体实验结构

```text
A. 无电机械测试
   ├─ 静态墙端力矩
   ├─ 插头拔出保持力
   ├─ 低能量扰动稳定性
   └─ 凹陷/遮挡插座可达性

B. 用户操作测试
   ├─ 30/60/90cm 线长比较
   ├─ 收纳时间与体积
   ├─ 地区线选择错误率
   └─ 状态提示理解度

C. 受控带电测试（实验室）
   ├─ AC 中断与 last-gasp 日志
   ├─ USB-C 热插拔和重新协商
   ├─ 端口功率分配
   ├─ 异常低功率提示
   └─ 温升与降额
```

## 3. 测试对象

### T0 — 现有长机身直插基线
约 65W 壁插 GaN 充电器，不叠加旅行转换头。

### T1 — 转换头叠加基线
地区转换头 + 约 65W 壁插 GaN 充电器。

### T2 — 稳定型短机身壁插
参考安克 Prime 67W 一类强调短机身和壁插稳定性的结构。

### T3 — Atlas Relay 等效样机
轻量地区插头 + 30/60/90cm 输入线 + 桌面等质量主机模型。初期机械实验不接市电。

## 4. 测试台模块

### R1 — 可倾斜插座安装板
- 插座面板角度：0°、30°、60°、90°；
- 可安装 Type A、Type C/F、Type G 等插座样本；
- 插座前方设置 0、10、20、30mm 可调凹陷框；
- 一侧设置家具遮挡板，模拟床头柜或桌板。

### R2 — 墙端载荷与位移测量
- 数字测力计：测量拔出保持力；
- 电子秤或已知质量块：校准墙端质量；
- 卡尺：测量重心到墙面的距离；
- 手机慢动作视频：记录角位移和脱落时刻。

### R3 — 低能量扰动装置
采用可重复摆锤，而不是手工敲击：
- 摆锤质量：100g；
- 摆长：300mm；
- 释放角度：5°、10°、15°；
- 每种结构每个条件重复 10 次。

摆锤只在无电机械模型上使用。

### R4 — 线材可达性场景箱
四个障碍：
1. 插座凹陷 30mm；
2. 插座与床头柜侧面距离 35mm；
3. 插座位于桌板后方 250mm；
4. 插座高于可放置平面 600mm。

比较直插 + 长 USB-C 线与 30/60/90cm AC 输入线。

### R5 — 受控供电与日志测试台
只在受监督实验室中使用：
- 经过认证的可编程 AC 电源或封闭式可控 AC 开关；
- 漏电和过流保护；
- USB-C PD 分析仪；
- 可编程 USB-C 电子负载；
- 示波器或高速记录器；
- 热电偶或红外热像仪；
- 阻燃、非导电工作台。

不得自行搭建裸露市电继电器盒。

## 5. 实验 A：静态墙端力矩与保持力

\[
\tau = \sum_i m_i g d_i
\]

记录：
- 初始拔出保持力 \(F_r\)；
- 插头端角位移；
- 是否完全脱落；
- 接触中断前的位移；
- 测试后的插脚和插座损伤。

条件：
- 插座角度：0°、30°、60°、90°；
- 插座样本：新、正常使用、低保持力替代样本；
- 结构：T0、T1、T2、T3；
- T3 线长：30、60、90cm；
- 每个条件至少 10 次重复。

主要比较：
- T1 vs T3：叠加转换头与轻墙端；
- T2 vs T3：短机身壁插是否已经足够；
- T3 不同线长：线材是否反向拉扯墙端。

## 6. 实验 B：扰动与断连阈值

无电机械阶段摆锤能量：

\[
E = m g L(1-\cos\theta)
\]

记录：
- 第几次冲击后出现明显位移；
- 第几次冲击后脱落；
- 最大稳定释放角度；
- 不同插座保持力下的差异。

带电连续性阶段不使用摆锤，改用：
- 线材横向拉力：0.2N、0.5N、1.0N；
- 人工插拔设备；
- AC 中断：50ms、200ms、1s、5s；
- 每种事件重复 20 次。

## 7. 实验 C：充电完整性检测

应检测：
- AC 输入消失与恢复；
- USB-C 端口断开；
- PD 合约重新建立；
- 其他端口是否发生非预期掉电；
- 实际输出长期低于协商功率；
- 过温与保护事件；
- 超级电容是否完成最后一次日志写入。

评价指标：

| 指标 | 定义 |
|---|---|
| Event Recall | 实际事件中被正确记录的比例 |
| False Alarm Rate | 正常事件被误报的比例 |
| Timestamp Error | 记录与参考仪器时间差 |
| Last-Gasp Success | AC 完全断开后成功写入日志的比例 |
| Cross-Port Dropout | 插拔一个端口时其他端口失电时长 |

工作目标：
- AC 中断 ≥200ms：事件召回率 ≥99%；
- 1s 以上中断：last-gasp 日志成功率 ≥99%；
- 正常 PD 重新协商误报率 ≤5%。

这些是设计目标，不是已达到的结果。

## 8. 实验 D：多口功率与温升

负载：
- 30W、65W、100W 单口；
- 65W + 35W；
- 45W + 30W + 25W。

环境：
- 25°C；
- 35°C；
- 每个稳态负载运行 2 小时；
- 记录外壳热点、关键点温度、降额和效率。

必须覆盖：
- 100V、120V、230V 输入；
- 多口插拔；
- 阻挡散热和软包邻近的误用场景；
- 过温保护恢复行为。

## 9. 实验 E：线长与用户操作

受试者：
- 12–20 名目标用户；
- 至少一半有跨国、多设备旅行经历；
- 包含反方和轻装用户。

任务：
1. 根据“德国 + 英国”行程选地区线；
2. 在四种障碍场景中完成安装；
3. 连接笔记本、手机和第三台设备；
4. 判断黄色状态所代表的事件；
5. 收纳产品。

指标：
- 安装成功率和时间；
- 地区线选择错误率；
- 线长满意度；
- 收纳体积；
- 状态解释正确率；
- 是否认为短线收益大于携带负担。

## 10. 统计分析

- 连续变量：均值、中位数、标准差、95% 置信区间；
- 脱落/断连：二项比例及置信区间；
- 同一插座结构比较：配对分析；
- 线长任务：重复测量设计；
- 预先写明排除规则；
- 原始数据不可手工覆盖。

## 11. Kill Criteria

1. T3 相对 T2 在低保持力插座中没有稳定性优势；
2. 60% 以上目标用户更偏好直插 + 长 USB-C 线；
3. 断充检测误报率长期高于 10%；
4. last-gasp 需要大容量电池才能可靠工作；
5. 三口独立不中断架构导致不可接受的体积、成本或温升；
6. 地区短线选择和收纳错误抵消机械价值。
